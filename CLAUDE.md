# Swerve

A 3D ball-rolling endless runner built with Three.js & Cannon-es. (MOBILE WEB: iOS & Android)

## Tech Stack

- **Three.js** — 3D rendering
- **Cannon-es** — Physics engine
- **Vanilla JavaScript** — No frameworks
- **HTML/CSS** — HUD overlay
- **GitHub Pages** — Hosting

## Project Status

**DONE:** Core gameplay mechanics, physics, HUD/UI, high score persistence, obstacle redesign (brick-textured walls, red X sliding blocks), neon yellow arch collectibles, speed boost collectible, ghost mode & hit slowdown system, 7-level difficulty scaling, skybox with aurora shaders, GPU shader warm-up, mobile performance optimization, and futuristic neon hexagon background.
